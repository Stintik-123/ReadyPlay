01
Telegram через tgwsproxy
VPS · 5 минут
+

tgwsproxy — легковесный WebSocket-прокси для Telegram. Поднимается на любом VPS, обходит DPI через маскировку под обычный HTTPS.

Что понадобится

  • VPS с белым IP (от 100₽/мес)
  • Домен (или работа через IP)
  • SSH-клиент

Шаг 1. Подключение к серверу

ssh root@YOUR_SERVER_IP

Шаг 2. Установка зависимостей

apt update && apt install -y git golang-go

Шаг 3. Сборка

git clone https://github.com/username/tgwsproxy
cd tgwsproxy
go build -o tgwsproxy

Шаг 4. Запуск

./tgwsproxy -listen :8443 -secret YOUR_SECRET

Шаг 5. Настройка клиента

В Telegram-клиенте укажи: Server — IP или домен VPS, Port — 8443, Secret — YOUR_SECRET.

02
Telegram через ByeByeDPI
Локально · 3 минуты
+

ByeByeDPI — форк zapret под Telegram. Работает без VPS, прямо на твоём компьютере или роутере.

Шаг 1. Скачивание

Релиз — с официального репозитория ByeByeDPI. Для Windows — byebyedpi-win.zip, для Linux — byebyedpi-linux.

Шаг 2. Запуск на Windows

От имени администратора:

byebyedpi.exe --demode --auto

Флаг --auto подбирает параметры под твой провайдер.

Шаг 3. Запуск на Linux

sudo ./byebyedpi --demode --auto

Шаг 4. Прокси в Telegram

ByeByeDPI поднимает SOCKS5 на 127.0.0.1:1080. В Telegram: SOCKS5, 127.0.0.1, 1080.

03
Discord и YouTube через zapret
Роутер / VPS · 10 минут
+

zapret (от bolvan) — универсальный обход DPI. Работает на уровне ядра, модифицирует пакеты на лету.

Шаг 1. Установка на OpenWRT/Keenetic

opkg update
opkg install git
git clone https://github.com/bolvan/zapret
cd zapret
./install_bin.sh

Шаг 2. Установка на Linux

git clone https://github.com/bolvan/zapret
cd zapret
./install_easy.sh

Шаг 3. Выбор стратегии

  • TPWS — легковесная, без root на клиенте
  • NFQWS — мощная, работает с QUIC и TLS
  • MDIG — обход блокировки по IP

Шаг 4. Списки

Отредактируй /opt/zapret/ipset/zapret-hosts-user.txt:

discord.com
discordapp.com
youtube.com
googlevideo.com
ytimg.com

Шаг 5. Применение

/opt/zapret/start.sh
systemctl enable zapret
